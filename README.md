# 🚀 SAW Controller (SAWC)
**Standalone Advanced Walk Controller** — это **ультимативное модульное решение** для перемещения персонажа в Unity.

---

## 📖 [ЧИТАТЬ ПОЛНУЮ ДОКУМЕНТАЦИЮ](https://qmrcl.github.io/SAWC-docs/)
> **Важно:** Перед использованием убедись, что у тебя установлены **Cinemachine** и **Input System**.

---

### 🔥 ГЛАВНЫЕ ФИШКИ (FEATURES)

* **⚡ Модульность:** Логика ядра (`Core`) полностью отделена от визуала и звука.
* **🛠 Scriptable Settings:** Все настройки скорости, прыжка и физики меняются через **один ассет**.
* **📱 Touch Support:** Умный тачпад с **адаптацией под DPI** и игнором джойстиков.
* **🪐 Advanced Physics:** Множитель падения (**Fall Multiplier**) для ощущения «тяжелого» и реалистичного прыжка.
* **🎮 Device Detector:** Автоматическое определение: Клава / Геймпад / Тач.

---

### 🛠 БЫСТРЫЙ СТАРТ (QUICK START)

1.  **Установка:** Подключи пакет через Package Manager (`Add package from disk...`).
2.  **Настройка:** В настройках проекта (`Project Settings`) поставь:
    * `Active Input Handling` ➔ **BOTH** (или **Input System Package**).
3.  **Префаб:** Перетащи префаб игрока на сцену и сделай **`Unpack Completely`**.

---

### 🏗 АРХИТЕКТУРА (ASMDEF)

Проект разбит на независимые сборки для **максимальной скорости компиляции**:
* `SAW.Core` — **Святая святых**. Только физика и ввод.
* `SAW.Modules` — **Обвес**. Камера, звуки, спецэффекты.
